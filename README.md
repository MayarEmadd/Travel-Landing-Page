# Travel Landing Page

A clean and modern **travel landing page** built using **HTML5 and CSS3**.  
This project showcases a beautiful layout for travel websites with a focus on simplicity, visuals, and user experience.


# Features

- Clean and modern UI
- Organized sections (Hotels, Destinations, Bookings, Contact)
- Lightweight and fast


# Technologies Used

- HTML5  
- CSS3
